# HTTP 메서드 활용

### 클라이언트에서 서버로 데이터 전송

1. 쿼리 파라미터를 통한 데이터 전송
    
    ⇒ GET, 주로 정렬 필터를 사용한다.
    
    사용처
    
    ⇒ 정적 데이터 조회 [ 이미지, 정적 텍스트 문서 ]  
    
    → 보통 쿼리 파라미터 없이 리소스 경로로 단순 조회
    
    ⇒ 동적 데이터 조회 [ 검색, 게시판 목록에서 정렬 필터 ]
    
2. 메세지 바디를 통한 데이터 전송
    
    ⇒ POST, PUT, PATCH
    
    → ex. 회원가입, 상품주문, 리소스 등록, 리소스 변경
